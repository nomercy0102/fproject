IP хаяг гэж юу вэ?

IP хаяг гэж юу вэ? Интернетэд холбогдсон компьютер бүр өөрийн IP хаяг гэгч тоон хаягтай байдаг. Тэр хаягаараа интернеттэй холбогдож харилцаж байдаг. Интернет дэх олон зуун тооцоолуур дотроос IP хаягаараа ялгарч тооцоолуурууд хоорондоо мэдээлэл дамжуулдаг. Таны веб сайт нэг тооцоолуур дээр байрлаад түүн рүү IP хаягаар хандаад нээж үздэг.

IP хаяг нь сүлжээний ID-ийн дагуу 5 төрөлд хуваагдана: A / B / C / D / E.

  1. Ангилал A: Энэ анги нь маш том сүлжээнд ашиглагддаг. Эхний октэт дэх 1-ээс 127 хүртэлх IP хаяг нь энэ бүлгийнх юм;
  2. Ангилал B: Энэ анги нь дунд хэмжээний сүлжээнд ашиглагддаг. Эхний октет дээр 128-с 191-ийн IP хаяг энэ бүлэгт хамаарна;
  3. В ангилал: Энэ анги нь жижиг ба дунд хэмжээний сүлжээнд ашиглагддаг. Анхны октет дэх 192-ээс 223 хүртэлх IP хаяг энэ бүлэгт хамаарна;
  4. Ангилал D: Энэ анги нь multicasting ашиглаж байна. Эхний гурав дахь орон нь 1, дөрөв дэх орон нь 0 байна.
  5. Ангилал E: Энэ анги нь зөвхөн туршилтын зорилгоор л зориулагдсан. Энэ нь D ангитай маш төстэй. Эхний гурав дахь битийн утга нь 1 байна. Зөвхөн ялгаа нь дөрөв дэх битийн утга нь 1 байна.
